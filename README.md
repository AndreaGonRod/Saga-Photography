# Saga API - Gestión para Fotógrafos

API REST con **Java**, **Spring Boot** y **PostgreSQL** para centralizar y automatizar el flujo de trabajo de un estudio fotográfico.

## Funcionalidades

* **Clientes:** Agenda, registro e historial.
* **Trabajadores:** Control de equipo (fotógrafos, editores) y roles.
* **Citas y Calendario:** Reuniones, consultas y tareas diarias.
* **Sesiones y Eventos:** Planificación de reportajes (bodas, estudio, etc.).
* **Flujo de Trabajo:** Seguimiento desde la captura hasta el postprocesado y entrega.
